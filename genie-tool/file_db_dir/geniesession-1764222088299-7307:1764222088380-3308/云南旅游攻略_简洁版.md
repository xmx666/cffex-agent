# 云南7天深度游攻略：昆明-大理-丽江-香格里拉
## 行程概览
云南是中国西南部的旅游胜地，拥有丰富的自然风光和民族文化。本攻略推荐一条经典的7天行程路线，涵盖昆明、大理、丽江和香格里拉四个城市，适合首次前往云南的游客。
- **第1天**：抵达昆明，游览石林
- **第2天**：昆明市区游览，前往大理
- **第3天**：大理洱海骑行，游览古城
- **第4天**：丽江古城，玉龙雪山
- **第5天**：虎跳峡徒步
- **第6天**：香格里拉普达措国家公园
- **第7天**：返回昆明，结束旅程
## 每日详细行程
### 第1天：昆明 - 石林
抵达昆明后，前往世界自然遗产——石林。石林以其奇特的喀斯特地貌闻名，是云南最具代表性的景点之一。
- **游览时间**：约3小时
- **交通方式**：打车或包车前往（约1小时车程）
### 第2天：昆明市区 - 大理
上午游览昆明市区，如翠湖公园、云南民族村。下午乘高铁前往大理。
- **游览时间**：上午昆明市区，下午大理
- **交通方式**：高铁（约2小时）
### 第3天：大理洱海骑行 - 大理古城
骑行洱海环湖公路，欣赏湖光山色。下午游览大理古城，感受白族文化。
- **游览时间**：全天
- **交通方式**：租电动车或自行车
### 第4天：丽江古城 - 玉龙雪山
上午游览丽江古城，下午前往玉龙雪山，体验高山雪景。
- **游览时间**：全天
- **交通方式**：包车或旅游专线车
### 第5天：虎跳峡徒步
徒步虎跳峡，感受峡谷的壮丽与险峻。
- **游览时间**：全天
- **交通方式**：包车前往
### 第6天：香格里拉普达措国家公园
游览普达措国家公园，欣赏高原湖泊和原始森林。
- **游览时间**：全天
- **交通方式**：包车或旅游专线车
### 第7天：返回昆明
从香格里拉返回昆明，结束旅程。
- **游览时间**：上午
- **交通方式**：飞机或高铁
## 主要景点介绍
### 石林
石林位于昆明市郊，是世界自然遗产，以其奇特的喀斯特地貌闻名。石林中的石柱、石峰、石笋等形态各异，令人叹为观止。
### 洱海
洱海是大理的标志性景点，湖水清澈，四周山峦环绕。游客可以骑行环湖，欣赏湖光山色。
### 丽江古城
丽江古城是世界文化遗产，以其独特的纳西族建筑和文化闻名。古城内小桥流水，古色古香。
### 玉龙雪山
玉龙雪山是丽江的标志性景点，海拔5596米，是云南最高的山峰之一。游客可以乘坐缆车登顶，欣赏雪山美景。
### 虎跳峡
虎跳峡是世界上最深的峡谷之一，位于金沙江上游。游客可以徒步穿越峡谷，感受大自然的壮丽。
## 美食推荐
云南的美食丰富多样，以下是一些值得尝试的特色美食：
- **过桥米线**：云南特色小吃，米线鲜嫩，汤头浓郁。
- **汽锅鸡**：用汽锅蒸制的鸡肉，肉质鲜嫩，汤汁鲜美。
- **野生菌火锅**：云南盛产野生菌，火锅中加入各种菌类，味道鲜美。
- **酥油茶**：藏族特色饮品，香浓可口，适合高原地区饮用。
## 旅行贴士
### 高原反应预防
云南部分地区海拔较高，建议提前服用红景天等药物，避免剧烈运动。
### 最佳旅游季节
云南四季如春，最佳旅游季节为3月至5月和9月至11月。
### 穿衣建议
云南昼夜温差大，建议携带保暖衣物和防晒用品。
### 注意事项
尊重当地民族风俗，注意环保，不乱扔垃圾。
## 实用信息
### 各城市间距离
- 昆明 - 大理：约300公里
- 大理 - 丽江：约200公里
- 丽江 - 香格里拉：约180公里
### 交通方式建议
- 昆明 - 大理：高铁（约2小时）
- 大理 - 丽江：高铁或包车（约3小时）
- 丽江 - 香格里拉：包车或旅游专线车（约3小时）
### 住宿推荐
- **昆明**：推荐住在市中心，方便出行。
- **大理**：推荐住在洱海边的民宿，方便骑行。
- **丽江**：推荐住在古城内，感受古城氛围。
- **香格里拉**：推荐住在松赞林寺附近的酒店，方便游览。
