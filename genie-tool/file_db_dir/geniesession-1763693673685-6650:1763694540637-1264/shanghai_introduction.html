经济与金融中心

上海是中国金融体系的核心，拥有上海证券交易所、中国金融期货交易所等国家级金融基础设施。陆家嘴金融贸易区汇聚了大量国内外金融机构，包括银行、证券、保险、基金和金融科技公司，形成了高度集聚的金融产业集群。上海自贸区的设立进一步推动了金融开放创新，成为人民币国际化和资本项目可兑换的重要试验田。

世界500强企业总部聚集

夜晚的外滩灯光璀璨，与对岸陆家嘴的霓虹交相辉映，是游客和市民必到的打卡地。

位于黄浦江东岸，是上海的金融中心和现代化都市形象的象征。这里矗立着东方明珠广播电视塔、上海中心大厦（中国第一高楼）、环球金融中心和金茂大厦等标志性建筑，共同构成了世界罕见的超高层建筑群。
