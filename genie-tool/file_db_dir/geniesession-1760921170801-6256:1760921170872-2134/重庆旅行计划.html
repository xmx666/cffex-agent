第一天：渝中半岛 · 城市脉搏与江岸夜景

洪崖洞

洪崖洞是重庆最具代表性的吊脚楼建筑群，依山就势，层层叠叠，夜晚灯火通明时宛如千与千寻中的奇幻世界。建议傍晚时分前往，可从1楼进入，沿阶梯逐层游览，感受巴渝传统民居与现代商业的融合。顶层观景平台可俯瞰嘉陵江与长江交汇处，是拍摄夜景的绝佳位置。

解放碑

解放碑全称“抗战胜利纪功碑暨人民解放纪念碑”，是重庆的地标性建筑，也是中国唯一一座纪念中华民族抗日战争胜利的纪念碑。周边是重庆最繁华的商业中心，步行街汇聚了国际品牌与本土老字号，是体验山城都市活力的核心区域。可在此品尝地道重庆小面，感受城市烟火气。

长江索道

长江索道是横跨长江的空中交通工具，被誉为“万里长江第一条空中走廊”。从南岸的上新街站乘坐，可俯瞰长江江面与两岸城市天际线，全程约5分钟。建议避开早晚高峰，选择白天前往，光线充足，视野开阔，是体验山城立体交通的独特方式。

第二天：沙坪坝与渝北 · 历史回响与城市奇观

磁器口古镇

磁器口古镇是重庆保存最完好的明清风格古镇之一，曾是嘉陵江畔重要的水陆码头，素有“小重庆”之称。青石板路、吊脚楼、老茶馆、手工作坊林立，可品尝陈麻花、毛血旺、酸辣粉等特色小吃。建议上午前往，避开人流高峰，静心感受古镇的慢节奏生活。

渣滓洞

渣滓洞原为一处小煤窑，后被国民党军统特务机关改造成监狱，是“中美合作所”集中营的重要组成部分。现为爱国主义教育基地，馆内陈列大量历史文献、实物与场景复原，真实还原了革命先烈在极端环境下不屈不挠的斗争精神。参观需保持肃穆，建议预留1.5小时以上。

李子坝轻轨穿楼

李子坝站是重庆轨道交通2号线的特色站点，轻轨列车在居民楼中穿行而过，被誉为“世界奇观”。观景平台位于小区旁的观景步道，可近距离拍摄列车“穿楼”瞬间。建议选择下午4-6点光线柔和时段，配合江景与山城背景，拍摄效果最佳。此地是重庆“立体交通”城市智慧的生动体现。

南山一棵树观景台
