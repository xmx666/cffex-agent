2. 贪吃蛇类实现

class Snake:
    def __init__(self):
        self.length = 1
        self.positions = [((WIDTH // 2), (HEIGHT // 2))]
        self.direction = random.choice([UP, DOWN, LEFT, RIGHT])
        self.color = GREEN
        self.score = 0

    def get_head_position(self):
        return self.positions[0]

    def move(self):
        head = self.get_head_position()
        x, y = self.direction
        new = (((head[0] + (x*10)) % WIDTH), (head[1] + (y*10)) % HEIGHT)
        if len(self.positions) > 2 and new in self.positions[2:]:
            return True  # 游戏结束
        else:
            self.positions.insert(0, new)
            if len(self.positions) > self.length:
                self.positions.pop()
        return False

    def reset(self):
        self.length = 1
        self.positions = [((WIDTH // 2), (HEIGHT // 2))]
        self.direction = random.choice([UP, DOWN, LEFT, RIGHT])
        self.score = 0

    def draw(self, surface):
        for p in self.positions:
            pygame.draw.rect(surface, self.color, (p[0], p[1], 10, 10))

游戏主循环是游戏运行的核心，负责处理用户输入、更新游戏状态和渲染画面。本实现中，我们使用pygame的事件队列来处理用户输入，通过clock.tick(FPS)来控制游戏帧率。

2. 碰撞检测

游戏中有两种碰撞检测：蛇与食物的碰撞(增加长度)和蛇与自身身体的碰撞(游戏结束)。通过比较坐标位置实现简单的碰撞检测。

3. 方向控制

通过限制不能直接反向移动(如向上移动时不能直接向下)来防止蛇"自杀"，这是贪吃蛇游戏的基本规则之一。

4. 边界处理

使用模运算实现"穿墙"效果，当蛇移动到屏幕边缘时会从另一侧出现，这是经典贪吃蛇的实现方式之一。
